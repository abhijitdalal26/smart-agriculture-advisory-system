1. Executive Summary 
This report covers the development and training of four machine learning models that form the core of the 
Smart Agriculture Advisory System — a Raspberry Pi-based precision farming assistant. The system 
integrates real-time soil and weather sensor data with trained XGBoost models to deliver actionable crop, 
yield, irrigation, and fertilizer recommendations. Three of the four models are production-ready. 
 
2. System Architecture & Hardware 
All models run inference on a Raspberry Pi loaded with a Flask REST API. Sensor readings are passed 
directly into the prediction pipeline, and the OpenClaw actuator responds to irrigation model outputs 
automatically. 
 
Hardware 
• Raspberry Pi — primary compute unit for inference 
• DHT22 sensor — temperature and humidity readings 
• Soil moisture sensor — real-time moisture percentage 
• Soil pH sensor — pH level monitoring 
• OpenClaw — configured and operational for automated irrigation actuation 
 
Software 
• Python 3.13  |  XGBoost, scikit-learn, pandas, numpy 
• Flask REST API serving all four model endpoints 
• Model persistence: .json (XGBoost native) + .pkl (encoders / metadata)
