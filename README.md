# Smart Agriculture Advisory System 🌾🚜

**An intelligent, IoT-powered, multi-model advisory system designed specifically for Indian farmers.**

The system integrates **real-time Raspberry Pi sensors** with **four dedicated XGBoost machine learning models** to provide precise, data-driven recommendations for:

- Crop Selection
- Yield Prediction
- Irrigation Scheduling
- Fertilizer & Soil Treatment

### ✨ Key Features

- Real-time sensor data (Soil moisture, NPK, pH, temperature, humidity, rainfall)
- Four independent XGBoost models (99%+ accuracy on Indian datasets)
- Hybrid intelligence (ML + rule-based decision engine)
- SMS alerts via GSM module (works in low-connectivity rural areas)
- Live web dashboard with trends and recommendations
- Fully modular and scalable architecture

### 🛠️ Tech Stack

- **Hardware**: Raspberry Pi + IoT Sensors
- **ML**: XGBoost + Scikit-learn
- **Backend**: FastAPI / Flask (Python)
- **Frontend**: React / HTML + Tailwind
- **Database**: SQLite / MySQL
- **Communication**: GSM module for SMS

### 📊 Datasets Used

- Crop Recommendation Dataset (Atharva Ingle)
- Agricultural Crop Yield in Indian States (Akshat Gupta)
- Irrigation Water Requirement Prediction Dataset
- Crop and Soil Dataset (Shankar Priya)

### 🚀 One-Line Summary

> A real-time IoT + XGBoost system that helps Indian farmers make smarter decisions and increase yield while reducing water wastage.
